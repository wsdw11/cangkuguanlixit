# 🚀 仓库管理系统 - 新手运行指南

## 第一步：安装 Node.js（如果还没有安装）

### 1. 下载 Node.js
- 访问官网：https://nodejs.org/
- 下载 **LTS 版本**（推荐，更稳定）
- 选择 Windows 安装包（.msi 文件）

### 2. 安装 Node.js
- 双击下载的 .msi 文件
- 一路点击"下一步"即可（使用默认设置）
- 安装完成后，**重启电脑**或**重启命令行窗口**

### 3. 验证安装
打开 PowerShell 或 CMD，输入以下命令：
```bash
node --version
npm --version
```
如果显示版本号（如 v18.17.0），说明安装成功！

---

## 第二步：安装项目依赖

### 1. 安装后端依赖
打开 PowerShell 或 CMD，进入项目目录，执行：

```bash
# 进入后端目录
cd backend

# 安装依赖（这可能需要几分钟）
npm install
```

### 2. 安装前端依赖
在**新的命令行窗口**中执行：

```bash
# 进入前端目录
cd frontend

# 安装依赖（这可能需要几分钟）
npm install
```

---

## 第三步：配置环境变量

### 创建后端配置文件

1. 在 `backend` 文件夹中，创建一个新文件，命名为 `.env`（注意：文件名就是 `.env`，没有其他名字）

2. 打开 `.env` 文件，复制以下内容进去：

```
PORT=3000
JWT_SECRET=my-warehouse-secret-key-2024
DB_PATH=./warehouse.db
```

**注意**：如果不会创建 `.env` 文件，可以：
- 在 `backend` 文件夹中创建一个文本文件
- 命名为 `.env.txt`
- 输入上面的内容
- 然后重命名为 `.env`（删除 `.txt` 后缀）

## 第四步：启动系统

### 方式一：使用两个命令行窗口（推荐）

#### 窗口1 - 启动后端服务
```bash
cd backend
npm run dev
```
看到 "服务器运行在 http://localhost:3000" 就说明后端启动成功了！

#### 窗口2 - 启动前端服务
```bash
cd frontend
npm run dev
```
看到类似 "Local: http://localhost:5173" 就说明前端启动成功了！

## 常见问题

### ❌ 问题1：提示 "node 不是内部或外部命令"
**解决**：Node.js 没有安装或没有添加到系统路径。重新安装 Node.js，并确保勾选"添加到 PATH"选项。

### ❌ 问题2：npm install 很慢或失败
**解决**：使用国内镜像源：
```bash
npm config set registry https://registry.npmmirror.com
```
然后再执行 `npm install`

### ❌ 问题3：端口被占用
**解决**：
- 后端端口 3000 被占用：修改 `backend/.env` 中的 `PORT=3001`
- 前端端口 5173 被占用：修改 `frontend/vite.config.ts` 中的 `port: 5174`

### ❌ 问题4：数据库错误
**解决**：删除 `backend/warehouse.db` 文件，重新启动后端服务（会自动创建新数据库）
